# BKW 0.9.0

Desktop GUI for BKW detonation, thermodynamic, and specific-impulse calculations

## Downloads

- **macOS (Apple Silicon):** `BKW-0.9.0-macos-arm64.dmg`
- **Windows (x64):** `BKW.exe`
- **Linux (x64):** `BKW-0.9.0-linux-x64.tar.gz`

## Highlights

- PySide6 desktop UI for preparing `BKWDATA`, running BKW/ISP calculations, and viewing/exporting results
- Built-in component and thermodynamic databases
- BKW, ISPBKW, USERBKW, and TDF engines run in-process for packaged app builds
